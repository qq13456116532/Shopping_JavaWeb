# shopping
简单的 javaWeb 项目(网店销售商城)

这只是一个简单的示例，你可以自己添加更多其他的功能，也可以美化一下这些页面。当然，应付一下老师应该还是勉强可以的。





#  使用方法

把 day19.sql导入到你的MySQL中，然后修改一下javaweb中的数据库配置，然后使用IDEA把这个项目导入就可以了。
